解法1：

```java
class Solution {
    public int findKthLargest(int[] nums, int k) {
        Arrays.sort(nums);
        return nums[nums.length - k];
    }
}
```

解法2：

```java
class Solution {
    public int findKthLargest(int[] nums, int k) {
        PriorityQueue<Integer> pq = new PriorityQueue<>();//小根堆
        for(int v : nums) {
            pq.add(v);
            if(pq.size() > k) {//维护堆的大小为k
                pq.poll();
            }
        }
        return pq.peek();
    }
}
```
