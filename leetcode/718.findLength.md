暴力解法：

```java
class Solution {
    public int findLength(int[] nums1, int[] nums2) {
        int m = nums1.length;
        int n = nums2.length;
        int ans = 0;
        for(int i = 0; i < m; i++) {
            for(int j = 0; j < n; j++) {
                if(nums1[i] == nums2[j]) {
                    int k = 1;
                    while(i + k < m && j + k < n && nums1[i + k] == nums2[j + k]) {
                        k++;
                    }
                    ans = Math.max(ans, k);
                }
            }
        }
        return ans;
    }
}
```

动态规划：

```java
class Solution {
    public int findLength(int[] nums1, int[] nums2) {
        //dp[i][j]：表示以i-1下标结尾的A和以j-1下标结尾的B，最长重复子数组为dp[i][j]
        //dp[i][j] = dp[i - 1][j - 1] + 1
        int m = nums1.length;
        int n = nums2.length;
        int ans = 0;
        int[][] dp = new int[m + 1][n + 1];
        for(int i = 1; i < m + 1; i++) {
            for(int j = 1; j < n + 1; j++) {
                if(nums1[i - 1] == nums2[j - 1]) {
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                }
                ans = Math.max(ans, dp[i][j]);
            }
        }
        return ans;
    }
}
```
