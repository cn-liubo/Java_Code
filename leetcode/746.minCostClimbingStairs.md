```java
class Solution {
    public int minCostClimbingStairs(int[] cost) {
        int n = cost.length;
        int[] dp = new int[n];
        dp[0] = cost[0];
        dp[1] = cost[1];
        for(int i = 2; i < n; i++) {
            dp[i] = Math.min(dp[i - 1], dp[i - 2]) + cost[i];
        }
        return Math.min(dp[n - 1], dp[n - 2]);
    }
}
```

优化之后：

```java
class Solution {
    public int minCostClimbingStairs(int[] cost) {
        int dp0 = cost[0];
        int dp1 = cost[1];
        int dpi = 0;
        for(int i = 2; i < cost.length; i++) {
            dpi = Math.min(dp0, dp1) + cost[i];
            dp0 = dp1;
            dp1 = dpi;
        }
        return Math.min(dp0, dp1);
    }
}
```
